# Frontend

The customer-facing web application.

- `src/app/` — routing, layouts, providers, and application initialization.
- `src/modules/` — domain-aligned customer experiences.
- `src/components/` — reusable, presentation-focused UI.
- `src/features/` — cross-module customer experiences.
- `src/lib/` — shared framework integrations and utilities.
- `src/styles/` — design tokens and global styling foundations.
- `src/assets/` — build-processed assets.
- `public/` — static assets served as-is.
