# Furniture Commerce Platform

Production-ready furniture e-commerce platform organized as a modular monolith. This repository intentionally contains structure and documentation only.

- `backend/` — server-side modular monolith.
- `frontend/` — customer-facing web application.
- `docker/` — container build and runtime assets.
- `docs/` — architecture and operational documentation.
- `.github/` — repository automation and contribution templates.
- `infra/` — infrastructure-as-code organization.

See [AGENTS.md](AGENTS.md) for engineering conventions.
